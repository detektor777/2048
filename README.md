# 2048 with Expectimax Solver

An advanced, mobile-friendly web implementation of the classic **2048** puzzle game, featuring an optimized **Expectimax Solver** and an interactive **Hint System**.



## [Play](https://detektor777.github.io/2048/)


* **Classic 2048 Gameplay**: Slide and merge matching tiles. The game does not end at 2048 and continues indefinitely.
* **High-Performance Solver**: Watch an optimized Expectimax algorithm solve the board in real time. It uses bitboard representations and precalculated heuristic tables for fast decision tree traversal.
* **Interactive Hint System**: Get real-time move recommendations from the algorithm while playing manually. The suggested direction is highlighted clearly to avoid confusion.
* **Responsive Design**: Fluidly optimized for both desktop monitors and mobile touch screens using modern CSS layout rules (`clamp` and `aspect-ratio`).
* **Autoplay Speed Control**: Adjust the delay slider from 0ms (maximum speed) up to 300ms.
